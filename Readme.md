# Wikipedia Live Edit Pipeline

A real-time streaming data pipeline that ingests live Wikipedia edit events 
and processes them through a structured pipeline architecture.

## What it does

- Connects to the Wikipedia EventStream API (Server-Sent Events)
- Filters and structures incoming edit events in real time
- Demonstrates stream ingestion, event filtering, and pipeline design patterns

## Stack

- Python
- Wikipedia EventStream / SSE
- Structured data processing

## Why this exists

Built to practice real-time data pipeline design — a core pattern in 
data engineering for systems that need to process high-velocity event streams.
